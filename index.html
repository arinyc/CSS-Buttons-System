An easy and elegant way of using buttons.

My name is Ari. I am a freelance web develoepr and UX designer based out of Williamsburg. I created this simple but effecient buttons system that will allow you my fellow web designers to save time by copying and pasting these elegant ready to go buttons.

Now I know how overwhelming web deisgn can get sometimes. So use this minimal system for your advantage. I will add more and more buttons with different styles, and effects.

This is a simple system that allows web designers to save time when working on creative projects. More elegant and creative buttons with effects and animations will be updated every week.


View Demo


How to use:

  • 1. Download the Zip file containing the elegantButtonsSystem CSS file.
  • 2. Extract the CSS files to the CSS folder of your project



    • Call the CSS file in your head tag like so:

     <link rel="css/elegantButtonsSystem.css" type="text/css" />

    Create a hyperlink

    <a href="">Elegant Button</a>

    or a button.

      <button>Elegant button</button>

    Add style.

    Buttons sizes

    X-Small.xsm Small.sm Medium .m Large.l


    Effects

    Shadow .shadow


    Corners

    Button corners .xsm-round .sm-round.m-round.l-round

    In this example I added the Extra Small button.
